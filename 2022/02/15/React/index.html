什么是 React

React是一个用于构建用户界面(html页面)的JavaScript库。

React 的三个特点

  1. 声明式

    • React 负责渲染UI，并在数据变化是更新UI

  2. 基于组件

    • 组件是react最重要的内容
    • 组件表示页面中的部分内容
    • 组合、复用多个组件，可以实现完整的页面功能

  3. 学习一次 随处使用

    • 开发web应用
    • 开发移动端原生应用(react-native)
    • 开发VR(虚拟现实) 应用(react 360)

React 的基本使用

  1. npm i react react-dom
  2. 引入 react 和 react-dom 脚本文件
  3. 创建 React 元素 let dom = React.creatElement("p",{title:"dyr"},"hello React")
  4. 渲染 React 元素到页面中 ReactDOM.render(dom,document.getElementById("root"))

React脚手架

推荐使用命令：npx create-react-app my-app
老版本 npm init react-app my-app
yarn create react-app my-app
启动项目：npm start

jsx 使用

jsx-JavaScript XML
优势：声明式语法更加直观、与HTML结构相同，降低了学习成本，提升开发效率

基本用法

const hello = <div> this is a react</div>

为什么脚手架中可以用jsx语法？

  1. JSX 不是标准的ECMAScript语法，它是ECMAScript的语法扩展
  2. 需要使用babel编译处理后，才能在浏览器环境中使用
  3. create-react-app 脚手架中已经默认有该配置，无需手动配置
  4. 编译JSX语法的包为：@babel/preset-react。

注意点

  1. 驼峰命名法
  2. 特殊属性：class——className、for——htmlFor、tabindex——tabIndex
  3. 没有子节点的React元素可以用 />结束
  4. 推荐：使用小括号包裹JSX，从而避免JS中的自动插入分号陷阱。

表达式

单大括号 {}
注意点：

  • 不能在{}中出现语句(比如 if/for 等 )
  • js中的对象是一个例外，一般只会出现在style属性中

jsx的条件渲染

  • if-else
  • 三元表达式
  • 逻辑与运算符

jsx的列表渲染

如果要渲染一组数组，应该用数组的map方法

jsx的样式处理

  • 类className (推荐)
    <div className="title">类样式</div>

  • 行内样式style/{}
    <div style={{color:'red',fontSize:'26px'}}>行内样式</div>

组件创建的两种方式

函数组件（无状态组件）

  • 首字母必须大写
  • 必须有return
  • 返回 null，不展示内容

class组件（有状态组件）

  • 首字母必须大写
  • 必须继承 React.Component
  • 必须要有 render() 方法
  • 必须有 return

React 事件处理

事件绑定

  • 与dom事件语法相似
  • on+事件名称 = {事件处理程序}，比如 onClick = {()=>{}}
  • 注意：驼峰命名法，比如：onMouseEnter、onFocus
  • 函数组件，无this，class组件，有this

组件的props

特点：

  • 可以给组件传递任意类型的数据
  • props是只读的对象，只能读取属性的值，无法修改对象
  • 注意：使用类组件时，如果写了构造函数，必须把props传给super(),否则无法在构造函数中获取props！
