VSCode常用设置

发表于2021-02-09|更新于2021-11-08
|阅读量:

VSCode语言设置

ctrl+shift+p ->  >configure Display Language -> en(英文)/zh-cn(中文)

VSCode基本设置

文件–首选项–设置，打开用户设置 ->

  1. editor.fontsize 用来设置字体大小，可以设置 editor.fontsize : 14;

  2. files.autoSave 这个属性是表示文件是否进行自动保存，推荐设置为onFocusChange——文件焦点变化时自动保存

  3. editor.tabCompletion 用来在出现推荐值时，按下Tab键是否自动填入最佳推荐值，推荐设置为on;

  4. editor.codeActionsOnSave 中的 source.organizeImports 属性，这个属性能够在保存时，自动调整 import 语句相关顺序，能够让你的 import 语句按照字母顺序进行排列，推荐设置为true, 即”editor.codeActionsOnSave”: { “source.organizeImports”: true }

  5. editor.lineNumbers 设置代码行号，即 “editor.lineNumbers”:”on”,

  6. editor.tabSize 设置tab值 ，即 “editor.tabSize”: 4,

  7. terminal.integrated.profiles.windows 集成git终端到vscode

  8. tabsize设置缩进

  9. format 保存格式化代码

插件

  1. Auto Close Tag 自动闭合 HTML/XML 标签

  2. Auto Rename Tag 自动完成另一侧标签的同步修改

  3. Bracket Pair Colorizer 给括号加上不同的颜色，便于区分不同的区块，使用者可以定义不同括号类型和不同颜色

  4. Debugger for Chrome 映射 vscode 上的断点到 chrome 上，方便调试

  5. ESLint js语法纠错

  6. Prettier - Code formatter 代码格式

  7. vetur 提供语法高亮和智能提示

  1. Beautify 代码美化工具， 该插件支持 js、json、css、sass、less和html 的代码美化,

  2. CSS Peek css快速预览与修改工具, 支持css的快速跳转

  3. Vue 2/3 Snippets 主要加强vue的便捷写法

  4. Document This 主要用于方法的注释，选中方法名，按两次 Ctrl+Alt+D,即可快速生成标准的注释

  5. JavaScript (ES6) code snippets 用于快速生成 ES6 代码片段

  6. path Autocomplete @的配置 先装插件,然后再 settings.json 中添加:

  7. Vue VSCode Snippets 插件: 在输入<时选 with default.vue 自动生成 vue 组件模板
